Java8新特性

发表于 2018-06-13 | 分类于 Java |

运用Java有一段时间了，前段时间了解了Java8的新特性，感觉简直就是非常棒的一次更新。。下面是我所学习Java8中的一些知识！！

1. Lambda表达式

Lambda表达式包含：（参数）->{正文}；

  1. 参数：参数为一个时无需加小括号，没有参数或需要参数都需要加小括号
  2. 箭头：->
  3. 正文：一条语句无需大括号，如需要写多条语句加上大括号；

接口只有一个方法可以写Lambda了。。。。。下面是重点。

2. Stream API（java.util.stream.*）

我们可以运用stream执行非常复杂的查找，过滤和映射数据等操作 。

注意：

  • stream本身不会储存元素。
  • 它不会改变原对象，

  • 操作是延迟的执行的，也就是说它们会等到需要的结果才会执行

    stream操作的三个步骤

  1. 创建stream流：一个数据源（如：集合，数组），获取一个流
  2. 中间操作：对数据源进行处理
  • filter：从流中排除某些元素；
  • limit：使其流不超过给定数量；
  • skip（n）：跳过n个元素，获取n个元素后面的元素，如元素不足n个，则返回一个空流；
  • distinct:筛选通过流所产生的元素的hashCode()和equals()去除重复元素；
  • map：将元素转换成其他形式或提取元素；
  • flatMap：接受一个函数作为参数，将流中的每个值都换成另一个流，然后把所有的流连接成一个流
  • sorted：排序
  • 其他：看文档；
  1. 终止操作：产生结果
  • allMatch：检查是否匹配元素；
  • anyMatch：检查是否至少一个元素；
  • noneMatch：检查是否没有匹配所有元素；
  • findFirst：返回第一个元；
  • findAny：返回当前任意元素；
  • forEach：循环
  • collect：转换

3. 其他

  1. java8新增了Optional：防止空指针
  • isPresent()方法判断是否为空；
  • get():获取值
  1. java8的时间API是JSR-310的实现，它的实现目标是克服旧的日期时间实现中所有的缺陷，特点：不变性，这对多线程环境有好处；
  • LocalDate.now(): 获取当前日期；格式：yyyy-MM-dd
  • LocalTime.now(): 获取当前时间；格式:HH:mm:ss
  • LocalDateTime.now()：获取当前日期时间 格式 yyyy-MM-dd HH:mm:ss

3.java8的接口支持已有默认实现方法：关键字：default ；

4. 总结
